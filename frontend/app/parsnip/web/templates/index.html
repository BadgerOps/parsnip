Review Limitations

Currently, the review checks for duplicate structures of the same type, duplicate fields/values within a specific structure, structures defined in multiple scopes, and checks for missing dependencies.

Frontend Limitations

Currently, there are several concepts that are supported by the Parsnip language, but are not currently implemented in the Frontend. Look at the syntax.md file in the docs folder of this repository for more information.
